# Local benchmark — 2026-07-23

This is a local regression run, not a publishable framework comparison. Each
fixture ran once in stable order on the same machine. Public results require
randomized order and at least five runs per scenario/concurrency combination.

## Configuration

- Duration: 30 seconds per fixture and scenario
- Connections: 64
- Load generator: `oha` 1.15.0, closed-loop
- Server artifacts: AOT executables
- Correctness: all HTTP fixtures and native RPC transports passed before load
- Machine: Apple M4, 16 GiB RAM
- OS: macOS 26.5 (25F71), arm64
- Dart: 3.11.0 stable
- Git base: `af24b1a`, with the uncommitted shared module-runtime DI changes

and `oha` reported a success rate of 100%. The 63–64 deadline aborts shown in
the raw files are the in-flight connections stopped when each fixed-duration

## Interpretation

`dart_orpc` led this single local run in throughput and p50/p99 latency for all
three shared HTTP scenarios. Compared with the next-highest-throughput fixture,
it delivered approximately 58% more plaintext requests/s, 66% more JSON
requests/s, and 55% more echo requests/s. These figures are useful as a local
regression signal only; CPU utilization and peak RSS were not captured, and the
stable fixture order can bias results.
